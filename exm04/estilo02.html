Estilos

A ordem que as folhas de estilo são aplicadas é, da menor para maior importância:

    1. Estilo do agente de usuário;
    2. Estilo do autor;
    3. Estilo do usuário;
  2. Estilo do autor por importância;
  3. Estilo do usuário por importância;

Seletores

Os seletores em CSS são operações de busca que retornam elementos HTML para serem estilizados. A ordem de importância que eles são resolvidos é:

  1. Importância (melhor evitar);
  2. Especificidade;
    • Cada id vale 100 pontos;
    • Cada classe vale 10 pontos;
    • Cada etiqueta vale 1 ponto;
  3. Posição no documento/cascata.
